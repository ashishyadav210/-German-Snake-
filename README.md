A fun twist on the classic Snake game where you play as Elvish Yadav and your food is a German Shepherd..!

This is a simple web-based game built with HTML, CSS, and JavaScript.

## Features
*   Classic Snake gameplay mechanics.
*   The snake's head is an image of **Elvish Yadav**.
*   The food is an image of a **German Shepherd**.
*   The game is easily customizable, allowing users to change the images for the snake and food.

## How to Play
1.  Open the `index.html` file in your web browser.
2.  Use the **arrow keys** (Up, Down, Left, Right) on your keyboard to control the snake.
3.  Guide the snake to eat the food to score points and grow longer.
4.  The game ends if the snake runs into the walls or its own body.

## How to Customize
You can easily change the snake and food images by editing the game's JavaScript file.
1.  Open the HTML file (e.g., `index.html`).
2.  Find the variables that store the image links for the snake head and the food.
3.  Replace the existing image links with your own direct image URLs.
4.  Save the file and reload the game to see your changes.
